# weather
Получение погоды для офисов в Лондоне, Шереметьеве и Череповце

# Установка
Вам понадобится установленный Python 3.6-3.9 и git

Склонируйте репозиторий
```bash
$ git clone https://github.com/valeriy131100/weather
```

Создайте в этой папке виртуальное окружение
```bash
$ python3 -m venv [полный путь до папки weather]
```

Активируйте виртуальное окружение и установите зависимости
```bash
$ cd weather
$ source bin/activate
$ pip install -r requirements.txt
```
# Использование
Находясь в директории weather исполните
```bash
$ bin/python main.py
```
Программа покажет погоду в трех офисах
